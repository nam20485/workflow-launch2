Interactive Development & Architecture Plan

System Architecture Overview

Profile Genie uses a service-oriented architecture orchestrated by .NET Aspire. This design ensures a clean separation of concerns: the Blazor UI for user interaction, a central Backend API for business logic, and a dedicated Playwright Service for robust browser automation.

Phased Development Plan

Sprint 0: Foundation

Goal: Establish a runnable, orchestrated solution.

  • Initialize Git repository & CI pipeline.
  • Create solution structure (.Web, .Api, .PlaywrightService).
  • Configure .NET Aspire AppHost.
  • Set up EF Core and initial database migration.

Sprint 1: Authentication

Goal: Implement the complete user account lifecycle.

  • Configure ASP.NET Core Identity.
  • Build Blazor UI for Login/Registration.
  • Implement JWT issuance and validation.
  • Add rate-limiting to login endpoint.

Sprint 2: Profile & Credentials

Goal: Allow users to manage their data securely.

  • Finalize DB models for profile data.
  • Build "My Profile" and "Settings" pages.
  • Implement credential encryption/decryption logic.

Sprint 3: Basic Automation

Goal: Prove the core scraping concept.

  • Create `/scrape` endpoint in Playwright Service.
  • Implement basic Playwright logic for login and data extraction.
  • Connect Backend API to the new service.

Sprint 4: Scout Agent Protocol

Goal: Implement the multi-page survey reconnaissance workflow.

  • Enhance `/scrape` to handle navigation ("Next" buttons).
  • Implement logic to compare discovered questions to user profile.
  • Generate and display "unknown questions" list in the UI.

Sprints 5-8: Execution & Polish

Goal: Build the final submission flow and harden the system.

  • Implement the `/fill` endpoint for final submission.
  • Build the "Application Assistant" UI.
  • Add bot detection avoidance measures (timing, fingerprints).
  • Implement CAPTCHA handling and beta test.

Core Technologies

.NET 9 & Aspire

The foundation for a modern, orchestrated, and cross-platform application.

Blazor Server

Enables a rich, interactive C#-based UI with a "desktop-like" feel.

Playwright

Provides reliable, modern, and powerful browser automation capabilities.

PostgreSQL

A robust, open-source object-relational database for storing all application data.

Entity Framework Core

The object-relational mapper (ORM) for seamless data access from C#.

ASP.NET Core Identity

Handles secure authentication, session management, and JWT-based API protection.
